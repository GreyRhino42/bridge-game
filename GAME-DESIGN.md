# 桥牌游戏化教学 - Game Design Document

## 核心理念
**游戏即教学，教学即游戏。**
玩家打开游戏，直接进入"故事模式"，在打牌的过程中自然学会桥牌。
没有枯燥的教程页面，只有越来越有趣的挑战。

## 玩家旅程

### 第一章：认识桥牌 (Level 1-3)
**Level 1: "你的第一墩牌"**
- 场景：你拿到13张牌，对手出了一张♠K
- 引导：高亮你的♠牌 → "你必须跟♠，选一张"
- 目标：理解"跟牌"规则
- 奖励：+10 XP, 🃏 徽章

**Level 2: "花色的力量"**
- 场景：你有♠A，对手出♠Q
- 引导：高亮♠A → "你有最大的♠！用它赢墩"
- 目标：理解大牌赢小牌
- 奖励：+15 XP

**Level 3: "将牌的秘密"**
- 场景：你没有♠了，但有♥将牌
- 引导：高亮♥牌 → "你没有♠了！用将牌♥吃掉它"
- 目标：理解将牌的作用
- 奖励：+20 XP, ⭐ 将牌大师徽章

### 第二章：学会叫牌 (Level 4-6)
**Level 4: "第一次开叫"**
- 场景：你拿到一手好牌(14HCP, 5张♥)
- 引导：分析面板显示HCP → "你有14点，足够开叫！"
- 目标：学会开叫1♥
- 奖励：+25 XP

**Level 5: "搭档的回应"**
- 场景：你开叫1♥，搭档加叫到2♥
- 引导：解释"搭档有3+张♥和6-9点"
- 目标：理解加叫的含义
- 奖励：+30 XP

**Level 6: "争叫的勇气"**
- 场景：对手开叫1♣，你有12HCP+5张♠
- 引导：分析面板 → "你可以争叫1♠！"
- 目标：理解争叫
- 奖励：+35 XP, ⚔️ 竞叫者徽章

### 第三章：约定叫 (Level 7-9)
**Level 7: "Stayman问叫"**
- 场景：搭档开叫1NT
- 引导：高亮2♣按钮 → "用Stayman问搭档有没有4张高花"
- 目标：掌握Stayman
- 奖励：+40 XP

**Level 8: "Jacoby转移"**
- 场景：搭档开叫1NT，你有5张♥
- 引导：高亮2♦ → "用转移叫让搭档叫2♥"
- 目标：掌握Jacoby Transfer
- 奖励：+45 XP

**Level 9: "Blackwood问A"**
- 场景：已配合♠，你有16HCP，想试探满贯
- 引导：高亮4NT → "用Blackwood问搭档有几个A"
- 目标：掌握Blackwood
- 奖励：+50 XP, 🏆 满贯猎手徽章

### 第四章：大师之路 (Level 10)
**Level 10: "自由搏击"**
- 场景：完整牌局，无提示
- 目标：独立完成叫牌+打牌
- 奖励：+100 XP, 👑 桥牌新手毕业徽章

## 引导系统设计

### 卡牌高亮
- 推荐出的牌有金色脉冲光环
- 不推荐的牌半透明
- 必须出的牌有蓝色边框

### AI搭档对话气泡
- 叫牌时：搭档的头像旁弹出气泡解释
- "我叫1♥表示我有5+张红心和12+点"
- "你加叫2♥很好！表示你有3+张支持"

### 每步引导
1. **暂停** - 游戏暂停，等待玩家理解
2. **高亮** - 用动画标出关键元素
3. **解释** - 简短文字说明（1-2句话）
4. **行动** - 玩家点击正确选项
5. **反馈** - 正确✓+鼓励 / 错误→解释为什么

## 技术实现

### 状态管理
```js
G.storyMode = true/false
G.currentLevel = 1-10
G.xp = 0
G.level = 1-8
G.badges = []
G.unlockables = []
```

### 关卡定义
每个Level是一个JSON对象：
```js
{
  id: 1,
  title: "你的第一墩牌",
  chapter: "认识桥牌",
  objective: "理解跟牌规则",
  setup: { hand: [...], trick: [...], trump: null },
  guidance: [
    { step: "highlight", target: "hand-S", text: "这是你的手牌" },
    { step: "highlight", target: "ts-N", text: "对手出了♠K" },
    { step: "prompt", text: "你必须跟♠，选一张出" },
    { step: "validate", action: "playSpade", correct: "any" }
  ],
  rewards: { xp: 10, badge: "🃏" },
  nextLevel: 2
}
```

### 引导层
- 独立于游戏逻辑的UI层
- 半透明遮罩 + 聚光灯效果
- 对话气泡 + 高亮动画
- 不修改游戏核心代码
