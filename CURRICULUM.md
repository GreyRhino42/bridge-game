# 桥牌刻意练习课程体系 - 108课

## 设计哲学
基于Anders Ericsson《刻意练习》核心原则：
- 每课一个具体目标（不是"学叫牌"而是"学会开叫1♥"）
- 即时反馈（出牌后立即告诉对错+原因）
- 渐进难度（从⭐到⭐⭐⭐⭐⭐）
- 弱点诊断（自动识别薄弱环节）
- 心理表征（建立专家级牌型模式库）

## 课程结构
每课包含：
1. **目标** - 一句话说明本课学什么
2. **讲解** - 2-3句核心知识点
3. **示范** - 一个标准示例
4. **练习** - 3道互动题（即时反馈）
5. **测验** - 1道综合题（评估掌握程度）
6. **XP** - 完成奖励

## 刻意练习系统特性

### 1. 弱点诊断引擎
- 追踪每课每题的正确率
- 自动识别薄弱环节
- 推荐针对性练习

### 2. 自适应难度
- 根据表现自动调整
- 连续正确→提高难度
- 连续错误→降低难度+复习

### 3. 心理表征训练
- 牌型模式识别（均型/非均型/半均型）
- 叫牌序列模式（开叫→应叫→再叫→成局）
- 打牌路线模式（飞牌/建立/投入/挤牌）

### 4. 进步可视化
- 5大阶段进度条
- 每课完成度
- 雷达图（叫牌/打牌/防守/约定叫/策略）
- 连续学习天数
- 总XP和等级

### 5. 间隔重复
- 错过的题7天后自动复习
- 薄弱知识点周期性重现
- 确保长期记忆
