# 桥牌新手教学系统 - 深度优化规范

## 愿景
将桥牌游戏从"能玩"升级为"能教"——让零基础用户通过游戏学会桥牌叫牌。

## 目标用户
- 完全不懂桥牌的新手
- 想学习叫牌策略的初级玩家
- 需要练习叫牌的中级玩家

## 核心需求

### [S1] 引导式新手流程 (Onboarding)
- 首次打开游戏时，显示欢迎引导（3-5步）
- 第一步：什么是桥牌？（30字简介 + 动画示意）
- 第二步：认识牌桌（标注四个位置和角色）
- 第三步：叫牌是什么？（简单解释叫牌的目的）
- 第四步：试试叫牌（引导用户完成第一次叫牌）
- 第五步：开始游戏！
- 引导完成后不再显示，localStorage记录

### [S2] 实时叫牌教练 (Real-time Coach)
- 叫牌时显示当前手牌分析：HCP、牌型、分布
- 推荐叫品 + 详细解释（为什么推荐这个）
- 显示"其他选项"和每个选项的理由
- 错误叫牌时显示"为什么不推荐"的解释
- 学习提示根据用户水平调整（新手/初级/中级）

### [S3] 叫牌场景库 (Scenario Library)
- 至少20个预设叫牌场景，按难度分级
- 每个场景：手牌 + 叫牌历史 + 正确答案 + 详细解释
- 分类：开叫、应叫、竞叫、满贯、防守
- 用户可以选择特定类别练习

### [S4] 叫牌分析仪表盘 (Analytics Dashboard)
- 显示用户叫牌准确率（总体 + 分类）
- 常见错误类型统计
- 进步趋势图（最近10次游戏）
- 推荐练习领域

### [S5] 交互式叫牌教程 (Interactive Tutorial)
- 6课完整教程，从零开始
- 每课：讲解 + 示例 + 互动练习 + 测验
- 课程进度保存
- 完成所有课程获得"叫牌新手"徽章

### [S6] 叫牌提示系统增强 (Enhanced Hints)
- 提示分级：简短提示 / 详细解释 / 完整分析
- 新手模式：每步都有提示
- 进阶模式：只在请求时显示
- 专家模式：关闭提示

### [S7] 游戏内学习面板 (In-Game Learning)
- 打牌时显示当前策略提示
- 每墩结束后分析出牌是否最优
- 显示"如果当时出X会怎样"的对比

## 技术约束
- 单HTML文件，零外部依赖
- 所有数据存储在localStorage
- 响应式设计（桌面+平板）
- 与现有游戏逻辑完全兼容

## 验收标准
- 新手能在5分钟内完成引导并开始第一局
- 叫牌提示准确率 > 90%
- 所有20个场景可正常运行
- 教程内容无错误
- 零JS错误
